CineMagic - Gestão de Bilheteira para Cinema

Este repositório contém o desenvolvimento de uma aplicação web para a CineMagic, uma empresa que organiza sessões de cinema e pretende digitalizar o processo de venda de bilhetes e controlo de acessos às sessões.

A aplicação foi desenvolvida com o Laravel Framework no back-end e utiliza Bootstrap/Tailwind CSS no front-end para garantir uma interface moderna e responsiva.
